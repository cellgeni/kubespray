COVID-19 Cell Atlas Data Ingestion

If you would like your data to be added to this website please email us using the following email address: cv19caingest@sanger.ac.uk

We accept all common data formats (AnnData, Seurat, SingleCellExperiment, Loom, csv/tsv, etc) as long as the object contains expression values and cell type annotations. Expression values can be in raw counts (which we will re-process) and/or normalised values (which we will serve directly).

For visualisation, we use cell embeddings (e.g. UMAP) included in the object if available, or we will re-generate from expression values, where batch information will be essential. Other metadata, whether technical or biological, such as location of sampling, donor id (anonymised if necessary), age, sex, etc, that helps interpretation is hugely welcome.

For unpublished data, we only show and allow retrieval of ~20 relevant genes (please refer to the list at this page "For unpublished datasets, the included genes are") until notified otherwise by the submitter. You can also opt to submit data of only the relevant genes, however in this case normalised expression values and cell embeddings are required.
